# Little Lemon - Système de Réservation

Ce projet est une application React de réservation de tables pour le restaurant Little Lemon.

##  Instructions d'installation

1. **Cloner le dépôt** : `git clone [URL_DU_DEPOT]`
2. **Installer les dépendances** : `yarn install`
3. **Lancer l'application** : `yarn start`
4. **Lancer les tests** : `yarn test`

## Fonctionnalités
- Formulaire de réservation avec validation en temps réel.
- Design entièrement responsive (Mobile, Tablette, Desktop).
- Accessibilité conforme aux normes WCAG (ARIA, Sémantique).
- Gestion des cas limites (dates passées, nombre de convives).
